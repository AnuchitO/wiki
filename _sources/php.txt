.. index:: PHAR
.. _phar:

PHAR
=====
| Wikipedia: `<https://en.wikipedia.org/wiki/PHAR_(file_format)>`__
| Docs: https://secure.php.net/manual/en/intro.phar.php
| Docs: https://secure.php.net/manual/en/phar.creating.php
| Docs: https://secure.php.net/manual/en/book.phar.php

PHAR (*PHP Archive*) is a packaging format for :ref:`PHP` programs.

- Compression: :ref:`Tar`, :ref:`Zip`, or PHAR
- Layout: Stub, Manifest, File Contents
- Metadata: "Meta-data can be any PHP variable that can be serialized."
- :ref:`Composer` and :ref:`PHPUnit` are distributed as PHARs (PHAR
  archives).
